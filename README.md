# Rails Nuxt.js アプリ

## デプロイ

Rails https://rails-app-api.herokuapp.com/

Nuxt.js https://nuxt-app-front.herokuapp.com/

## セットアップ

$ make setup

### 起動

$ make up

### 終了

$ make down

## 環境

### Rails

Ruby 2.7.5

Rails 7.0.0

PostgreSQL 14.1(local), 13.5(heroku)

http://127.0.0.1:3000/

### Nuxt.js

Node 17.3.0

Nuxt.js 2.15.8

http://127.0.0.1:8080/
