# Rails アプリ

## バージョン

Ruby 3.0.2

Rails 7.0.0

PostgreSQL 14.1(local), 13.5(heroku)

Node 17.3.0

Nuxt.js 2.15.8

## セットアップ

$ make setup

### 起動

$ make up

Rails コンテナ入り方

$ make rails

PostgreSQL コンテナ入り方

$ make pg

Nuxt.js コンテナ入り方

$ make nuxtjs

### 終了

$ make down
